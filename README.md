# Next-Level Product Sprint 2026

A modern, visually stunning hackathon landing page built with HTML, CSS, and Three.js.

## Features

- 🎨 Modern gradient animations
- ✨ Three.js particle effects
- 📱 Fully responsive design
- 🚀 Smooth scroll animations

## Deployment

This project is ready for deployment on any static hosting platform:

### Vercel
```bash
npm i -g vercel
vercel
```

### Netlify
```bash
npm i -g netlify-cli
netlify deploy --prod
```

### GitHub Pages
1. Go to Repository Settings
2. Navigate to Pages
3. Select "main" branch as source

## Technologies

- HTML5
- CSS3 (Custom properties, Grid, Flexbox)
- Three.js (r128)
- Google Fonts (Bebas Neue, Syne, DM Mono, Outfit)
